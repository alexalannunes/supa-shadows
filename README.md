## Supa Shadows

A box shadow generator :tada:

### Made With

* :sparkles: Next.js
* :heavy_check_mark: TypeScript
* :hammer: Chakra-UI
* :wrench: Jotai
* :books: highlight.js
* :heavy_plus_sign: react-colorful
___


### Features

* Add infinite shadows
* Edit element
* Sharable link
* Preserve state after refreh
* Examples (in progress)
